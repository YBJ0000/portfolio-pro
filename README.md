# portfolio-pro

Rebuild of the personal portfolio (see `REFACTOR-PLAN.md`).

## Scripts

- `npm run dev` — local dev server
- `npm run build` / `npm run start` — production build & serve
- `npm run lint` — ESLint
- `npm run format` — Prettier

## Environment

Copy `.env.example` to `.env.local` when connecting Clerk, Sanity, or Neon.
